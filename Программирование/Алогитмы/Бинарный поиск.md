Бинарный поиск - это алгоритм; на входе он получает отсортированный
список элементов (позднее я объясню, почему он должен быть отсортирован).
Если элемент, который вы ищете, присутствует в списке, то бинарный
поиск возвращает ту позицию, в которой он был найден. В противном случае
бинарный поиск возвращает null.
Яркий пример попытка угадать число от 1 до 100( сначала 50 потом 75 и т.д)

def binary_search(arr, item):  
    low = 0  
    high = len(arr) - 1  
    while low <= high:  
        mid = (low + high) // 2  
        gues = arr[mid]  
        if item == gues:  
            return mid  
        elif gues > item:  
            high = mid - 1  
        else:  
            low = mid + 1  
    return None
